### README (Deutsch)
__PDEL Theme Facelift für Racing League Tools__

Dies ist ein Add-On für das PDEL Facelift Theme für Racing League Tools.
Mit diesem Add-On habt Ihr Zugriff auf alle Teamlogos + das Ligalogo. (Stand: 08.05.2026)


**Installation**
Lade ALLE Dateien herunter.
Entpacke das Add-On.
Kopiere die Dateien in dein entsprechendes Verzeichnis von Racing League Tools.
